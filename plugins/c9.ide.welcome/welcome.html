The Triton IDE Minimal Editor Layout

While in this layout the buttons for several handy panels are hidden from view by default. You can gain access to these panels using key bindings. The list below are the most commonly used key bindings. For an exhaustive list visit the key bindings editor in the preference panel. You can also use the menus in the toolbar to find commands by walking throught the sub menus.

Key Command
Command-U Show and Hide the File Tree
Command-E Show and Hide the Navigate Panel
Command-Shift-E Show and Hide the Outline Panel
Command-, Show and Hide the Preference Panel
Command-F Show the Find and Replace Panel
Command-Shift-F Show the Find in Files Panel
